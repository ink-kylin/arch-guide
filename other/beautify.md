
## 主题


[光标主题 ](https://wiki.archlinuxcn.org/wiki/%E5%85%89%E6%A0%87%E4%B8%BB%E9%A2%98)


Bspwm部分窗口无法应用光标主题，将下列命令添加至bspwm文件

```sh
xsetroot -cursor_name cursor_name
```

polybar可以高度定制化，config.ini配置主题

```sh
[bar/example]
cursor = cursor-name

```
